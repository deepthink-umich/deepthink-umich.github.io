# DeepThink Lab Website

This repository contains the static website for DeepThink Lab at the University
of Michigan.

## Maintainer Notes

Preview locally from the repository root:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`. Use another port if `8000` is unavailable.

⭐ The site is static HTML, CSS, JavaScript, and image assets. CSS changes may
not appear locally because browsers cache stylesheets for faster loading. A hard
refresh forces the browser to reload those files: use `Ctrl+Shift+R` on
Windows/Linux or `Cmd+Shift+R` on macOS.
